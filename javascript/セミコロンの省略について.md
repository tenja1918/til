
### return, throw, break, continue, ++, -- (Restricted Production)
これらのトークンでは改行にセミコロンが挿入される。
```javascript
// これは [ 1, 2, 3] が返る
(()=>{
  return [
    1,
    2,
    3
  ];
})();

// これは undefined が返る
(()=>{
  return      // 改行があるのでセミコロンが付与されて return; と解釈される
  [
    1,
    2,
    3
  ];
})();
```

```javascript
function f(n) {
  return (typeof n === 'number') ? n * 10 : 0;
};

function g() {
  let b = 5 + f                   // 改行がなく、直後の()を関数呼び出しと解釈してしまう

  ((function(){ return 200; })());
};

console.log(b);     // 2005
```
